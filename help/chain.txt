El shell contiene operadores para encadenar comandos como son:
command1 ; command2 : Con este operador se pueden ejecutar varios comandos en una sola línea, uno a 
uno en el orden en que aparecen.
command1 && command2 : Este operador permite poner varios comandos unidos de forma que el comando n 
se ejecuta solo si el comando n-1 tiene un exit status satisfactorio.
command1 || command2: Funciona como operador lógico or, si el primer comando es satisfactorio no 
hace falta ejecutar el resto de comandos.
